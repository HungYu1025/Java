# Java Reservation system

本程式模擬預約排程系統，用戶只要簡單的幾個操作步驟就能完成預約。

## 設計步驟

### 1. 用戶登入/註冊

- 登入: 用戶輸入帳號和密碼。
- 註冊: 用戶提供基本資料（如姓名、電子郵件、電話號碼）來創建新帳號。

### 2. 選擇日期和時間

- 用戶選擇希望的日期和時間。

### 3. 選擇服務

- 用戶從系統提供的服務列表中選擇需要預約的服務

### 4. 確認預約

- 系統顯示預約信息的摘要，供用戶確認。
- 用戶確認後，系統完成預約並生成預約確認號碼。

### 5. 管理預約

- 管理者可以查看、取消或重新安排預約。
